# COVID-19 Web Apps

## Getting started

```bash
npm install
npm run dev

```

## Publish

```bash
# Build for pre environment
npm run build:pre

# Build for production environment
npm run build

```

## Doc

### Language

- Language Supported
  - English(en)
  - Chinese(zh)

- To Switch Language: LOCALE (default en) (config at /src/config/index)
